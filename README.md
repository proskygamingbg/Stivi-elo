# Stivi-elo

Simple static site for tracking CS2 (Faceit-style) ratings. Add players with nickname, position, rating and photo. Levels are configured in js/app.js.

To publish with GitHub Pages:
1. Go to Settings → Pages and select the `main` branch and `root` folder.
2. Save and visit the provided URL.

Files added:
- index.html
- css/styles.css
- js/app.js

Edit `js/app.js` to change level thresholds.
